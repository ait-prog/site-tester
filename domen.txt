**Пример для бесплатного размещения на GitHub Pages:**
1. Зарегистрируйтесь на github.com.
2. Создайте новый репозиторий, загрузите туда все файлы сайта.
3. В настройках репозитория включите GitHub Pages (Settings → Pages → Source: main branch).
4. Получите ссылку вида: https://ваш-логин.github.io/имя-репозитория/
5. Если хотите свой домен — купите его и укажите CNAME-запись на адрес GitHub Pages.
